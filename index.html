About me

I'm Melissa and I'm a Web Developer,
UX Designer

Enthusiastic and a passion to learn new things! I am an ER nurse turned UX Designer and Web Developer. I discovered Lambda School's UX Design program while stuck in bed, nursing a broken leg. I loved that UX Design encompassed some of the same aspects of nursing: empathy, psychology, and research. But, it wasn't enough for me. I wanted to learn to code so I could be a true unicorn and a have a complete understanding of the projects I work on!
